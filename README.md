# Work Hours Tracker

A web application for tracking work shifts, calculating working hours, and estimating earnings based on an hourly rate.

This project is built for learning full-stack development fundamentals, including JavaScript, API design, and database integration.

---

## ✨ Features

- Add work shifts (date, start time, end time)
- Set hourly rate per shift
- Automatically calculate total hours
- Automatically calculate earnings per shift
- View list of all shifts
- Calculate total hours and total earnings

---
